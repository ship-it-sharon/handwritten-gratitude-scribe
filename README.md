# Handwritten Gratitude Scribe

A fresh start. Planning in progress.

**The idea:** an app that takes the burden out of writing thank-you notes.
Generate heartfelt, personalized notes in your own handwriting style (or the
closest match), and have them printed and mailed for you — for weddings,
showers, birthdays, and every occasion where gratitude is expected but time
is short.
